    AffineMaps.AffineStep — Type
    struct AffineStep

    Represents a multiplicative or additive step in an affine transformation or it's inverse.

    Multiplicative step are left-applied, additive steps are right-applied:

    AffineStep(*, k)(x) ≈ k * x
AffineStep(+, k)(x) ≈ x + k
AffineStep(\, k)(x) ≈ k \ x
AffineStep(-, k)(x) ≈ x - k

    AffineStep(op, k) supports InverseFunctions.inverse and ChangesOfVariables.with_logabsdet_jacobian.
